## Implementation

This application uses only TypeScript without frameworks and third-party libraries. The structure of the index.html file has not changed.


## Running

To compile the TypeScript in this project, install the dependencies and run the `build` script.

```
# from examples/typescript
$ npm install
$ npm run build
$ open index.html
```
